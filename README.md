# Coconut Catcher

A simple **Tkinter** game where you catch falling coconuts with a basket.  
Created with Python and AI assistance.

## How to Play

- Move the basket left or right using the **arrow keys**.
- Catch the falling coconuts.
- If a coconut hits the ground, the game ends and a broken coconut appears.
- Press **Play Again** to restart.

## Requirements

- Python 3.x
- Tkinter
- Pillow library (`pip install pillow`)

## File Structure

- `CoconutCatcher.py` (main game code)
- `img/` (folder containing coconut, basket, and broken coconut images)
- `.gitignore` (to ignore virtual environment)
- `README.md` (this file)

## Notes

- The game keeps score and gradually increases falling speed.
